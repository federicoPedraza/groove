# Integrations: Jira and OpenCode

## Jira integration
Jira can connect with API token settings, pull assigned issues, and map issue priority into workspace tasks. Sync keeps local tasks aligned with external tracking.

## Jira connection flow
Configure site URL and account email, connect with token, then run sync. If connection fails, verify URL format, credentials, and token scopes.

## Sync behavior
Sync imports new issues and updates existing task entries. Review task title and description after sync to keep execution quality high.

## OpenCode settings
Global settings include OpenCode enablement, default model, and settings directory. These values tune how Groove launches and cooperates with OpenCode workflows.

## Practical tip
Use Jira for source-of-truth issue tracking and Groove tasks for execution context inside active worktrees.

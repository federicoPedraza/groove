# Navigation and Shortcuts

## Purpose
Groove navigation is optimized for quick movement across dashboard, worktrees, tasks, diagnostics, and settings.

## Primary navigation
Use page-level navigation to switch context quickly. Keep the active workspace loaded before running worktree or task actions.

## Keyboard leader shortcuts
Global settings expose keyboard shortcut leader and leader bindings. Customize bindings to match your team workflow without changing feature behavior.

## Fast help usage
Open Help for focused how-to questions when you are blocked in a flow. Ask with feature context, expected outcome, and constraints.

## Reliability tip
After heavy navigation and command usage, refresh state before recovery operations so actions run on current workspace data.
